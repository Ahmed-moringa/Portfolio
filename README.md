# Portfolio

This is my Portfolio for the projects I have completed. Kindly look through it and send feedback.

[![Portfolio-screenshot.png](https://i.postimg.cc/fTg2tr9k/Portfolio-screenshot.png)](https://postimg.cc/VdnB244c)

## Technologies Used

- HTML
- CSS (Bootstrap)


## Setup Instructions and Installation

- Clone this repository to a location in your file system. 'https://github.com/Ahmed-moringa/Portfolio.git'
- Open terminal command line then navigate to the root folder of the application.'cd Portfolio'
- Open 'index.html' on your Browser.
